# pdfextractor
# Simple PDF Extractor Program that Extract PDF to a file

## Usage :   
  ```
  --pages [PAGES], -p [PAGES]
  Specify Total Pages
  
  --name NAME, -n NAME  
  Name of PDF
  
  --output OUTPUT, -o OUTPUT
  Output Name
  ```
